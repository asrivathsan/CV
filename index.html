Bioinformatic packages/software

ONTbarcoder: https://github.com/asrivathsan/ONTbarcoder A tool for rapid DNA barcoding using MinION sequencer (published as Srivathsan et al. (2021), BMC biology). Includes a GUI to improve accessibility to MinION barcoding, allows for large scale barcoding in any operating system and can run on laptops. Shown to handle >8000 products, performs several quality control steps for barcodes in any MinION run.

miniBarcoder: https://github.com/asrivathsan/miniBarcoder Older software for DNA barcoding using MinION (published as Srivathsan et al. (2018), Molecular Ecology Resources). A command line tool that allowed for barcoding >3000 COI products from a single flow cell (cost per barcode <0.5 USD). This contained several error correction steps given the higher error rates of nanopore sequencing, particularly prior to development of high accuracy basecalling.

obj_cluster: A tool for clustering of DNA barcodes to mOTUs which reimplements the method of Meier et al. (2006) and allows for clustering at all possible thresholds and generating visualizations as dendrograms. It also allows for parsing and mapping of BLAST outputs onto the dendrogram to visualize the matches. Includes a GUI and is under development to include features that aid integrative taxonomy

readsidentifier: https://github.com/asrivathsan/readsidentifier-1.0 Pipeline for read-based identifications of Next Generation Sequencing data included in metagenomic pipeline published Srivathsan et al. (2015) Molecular Ecology Resources.

ngs-barcoder: A tool for rapid processing of data generated for Illumina based barcoding studies. It is routinely used and part of “NGS barcoding” pipeline described by Meier et al. (2016), Cladistics. Recently updated with parallelization and improved GUI.

plate_generator : A GUI for generating demultiplexing sheets for NGS barcoding experiments. It allows the user to select range(s) of positions in a plate, provide sample and experiment information, and images relating to the experiment that are then combined by the software to generate a demultiplexing sheet.

specimen_sorter: A GUI that displays where each specimen within a mOTU is on the 96-well plate. This enables selection of specimens that can be grouped physically into putative species units.
